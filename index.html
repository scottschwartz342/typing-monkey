Background

The infinite monkey theorem states that a monkey hitting keys at random on a typewriter keyboard for an infinite amount of time will almost surely type any given text, including the complete works of William Shakespeare. Let's see how many words our “monkey” can type in a given amount of time.

Let's get this monkey going...

Simply type in the amount of time (in seconds) you want the monkey to type for. Then hit the send button. If you would like to start again, you can rehit the send button.
